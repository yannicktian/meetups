# 2026: The Year of Harnesses

Interactive slides for the **Developer AI.xperience** meetup — Marseille, April 9, 2026. Hosted by Qalico × Gojob.

A single-page Next.js app serving as bold, keynote-style slides with embedded mockups, animated architecture diagrams, streaming UI simulations, and code reveals.

**Live:** https://meetups-talk.vercel.app

## The Talk

"2026 is the year of harnesses" — Claude Code is the canonical example, Mastra named the primitive, every domain that has a product can have a harness.

1. **Intro** — Who uses Claude Code? The harness engineering trend
2. **Alpha** — Gojob's recruiter AI, 3M conversations in production
3. **SaaS** — Selling Alpha, onboarding pain, the bottleneck
4. **In-App Agent** — Mastra + RAG + MCP + streaming UI
5. **Harness** — Ernest (48h hackathon), the six pillars, real code
6. **Vision** — Death of menus, intent first, build your own
