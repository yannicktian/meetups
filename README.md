# AI.xperience Meetup Slides

Interactive presentation for the **Developer AI.xperience night with Gojob, Volta Medical** meetup (April 10, 2026).

A single-page Next.js web app serving as bold, interactive slides — embedded mockups, animated architecture diagrams, streaming UI simulations, and code reveals.

## The Talk

"From 3M conversations to SaaS to in-app agents — the evolution of AI at Gojob"

1. **Alpha** — Recruiter AI assistant, 3M conversations in production
2. **SaaS** — Selling Alpha to France Travail, Persol, and more
3. **In-App Agent** — Self-serve onboarding with Mastra + AG-UI + MCP
4. **Future** — AI-native products and the harness pattern

## Stack

Next.js 16 | React 19 | Tailwind 4 | Framer Motion | Shiki | Vercel
